Stručný popis hry:
Jednoduchá implementace klasické hry Snake (Had) v Pythonu s využitím knihovny Pygame. Projekt zahrnuje postupné vytváření hry od základního okna až po plnou funkčnost. Aktuální verze (Fáze 5 - snake10_5.py) kompletně implementuje pohyb hada v mřížce ovládaný šipkami, řízení rychlosti pomocí časovače, generování jídla a zvětšování hada při jeho pojídání.
