# teststudent

1, Hra Snake, 22.02.2026

Stručný popis hry:
Jednoduchá implementace klasické hry Snake (Had) v Pythonu s využitím knihovny Pygame. Projekt zahrnuje postupné vytváření hry od základního okna až po plnou funkčnost. Aktuální verze (Fáze 6 - snake10_6.py) kompletně implementuje pohyb hada v mřížce ovládaný šipkami, řízení rychlosti pomocí časovače, generování jídla, zvětšování hada při jeho pojídání, detekci kolize s okrajem (reset hry) a detekci kolize hada se sebou samým (konec hry).
